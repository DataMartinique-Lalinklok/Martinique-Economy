#Data Analysis About Martinican Economy
Martinique represent an interesting case for study for the relation between mass energy economics balance. 
Martinique does not have much natural ressources for modern industry. Current technical system is based on key technology including rare earth mineral, oil and gas.
Can a countries with few ressources perform a peaceful simplification? Meaning make sure quality of life is still relevant, ensuring key commodities flowing, regenerating ecosystem in a world where there are jeoperdize.
This simplified economy analysis is based on import-export Jane Jacobs framework and the basis of ecology economy. It aims to picture the current states of martinicans economies, and the ripples from the past.
